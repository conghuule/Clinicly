# Clinicly - Backend

## About the project

RESTful API for clinic management website

## API Documents

> https://clinicly.fly.dev/docs

### Layout

```tree
├───controllers
├───docs
├───middlewares
├───models
├───routes
├───types
└───utils
```

A brief description of the layout:

- `controllers` contains files for the controllers.
- `docs` API documents generated by Swagger.
- `middlewares` contains all the middleware that have been created.
- `models` contains files that is related to the model.
- `routes` contains files for defining routes.
- `types` contains all the type that have been created.
- `utils` contains all utility functions.
